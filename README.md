# vxe-core

Vxe UI v3.x 核心库.

## 使用

```shell
npm install @vxe-ui/core
```

```javascript
// ...
import VxeUI from '@vxe-ui/core'
// ...

VxeUI.setConfig({
  // ...
})
```
